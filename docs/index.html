Test APIs offline, without the cloud

Lisek is a desktop API client for HTTP, GraphQL, WebSocket, SSE, and gRPC. Run collections, mock local endpoints, decode JWTs, and keep everything on your machine.

Everything you need to explore APIs

A focused alternative to cloud-based clients — fast, private, and fully local.

HTTP & REST

All methods, auth, cookies, form-data, JSONPath, binary preview, response diff, and cURL import/export.

GraphQL

Query editor, variables, introspection, and WebSocket subscriptions.

WebSocket & SSE

Live WebSocket log and Server-Sent Events panel for streaming APIs.

gRPC

Import .proto from disk or URL, reflection browser, unary and streaming RPCs.

Collections & Runner

Nested folders, CSV/JSON data-driven runs, scheduled requests, and Git folder sync.

Mock Server

Local HTTP stub with JSON, text, or file responses. Edit routes and test in the browser.

Import / Export

Postman, OpenAPI, Insomnia, Bruno, HAR, and workspace backup — all local.

Plugins

Base64, URL encode, SHA hashes, JWT decode, and AES crypto tools in the sidebar.

Screenshots

Collections, protocols, environments, scripts, and history — in one compact UI.

What’s new in v1.6.2

  • App Help — full workspace guide via F1 or the ? button in the title bar
  • Update notifications — optional alert when a newer GitHub release is available
  • Side-by-side workspace — address bar on top; request and response panes side by side
  • JSONPath panel — resizable result area, Help button, and save query results to environment variables
  • Import cURL into a collection — right-click a collection and paste a curl command
  • Tabs — middle-click to close; Close all button
  • Editor fixes — caret no longer jumps while typing in URL, tags, and variable fields
  • Teal theme — refreshed primary accent color

Download

Lisek runs on Windows 10/11 and Linux (x64). Pick a build below.

Windows

Installer or portable zip for Windows 10/11 (x64).

Lisek-Setup.exe Lisek-win.zip

Linux

AppImage (run directly) or tar.gz archive for Linux x64.

Lisek.AppImage Lisek-linux.tar.gz
All releases Build from source

If Windows shows “Windows protected your PC”, click More info, then Run anyway. Lisek is open source and not code-signed yet. On Linux, make the AppImage executable: chmod +x Lisek.AppImage
