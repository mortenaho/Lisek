Lisek

Offline API client for HTTP, GraphQL, WebSocket, and gRPC.

Download v1.6.4 View source

What it does

Every protocol you need, kept on your machine

No account. No sync cloud. Build collections, mock local routes, and ship requests without leaving your desk.

HTTP & REST

Auth, cookies, form-data, JSONPath, binary preview, response diff, and cURL import/export.

GraphQL

Queries, variables, introspection, and live subscriptions over WebSocket.

WebSocket & SSE

Streaming panels for sockets and Server-Sent Events.

gRPC

Proto import, reflection browser, unary and streaming RPCs.

Collections

Nested folders, smart search, scheduled runs, and Git folder sync.

Mock & tools

Local mock server, Postman/Bruno/OpenAPI import, JWT and crypto plugins.

Install

Free under MIT. Builds ship from GitHub Releases.

Windows

Installer or portable zip for Windows 10/11 (x64).

Lisek-Setup.exe Zip

Linux

AppImage, RPM, or tar.gz for Linux x64.

AppImage RPM tar.gz

Windows SmartScreen may warn until the app is signed — choose More info, then Run anyway. AppImage: chmod +x Lisek.AppImage. RPM: sudo dnf install ./Lisek.rpm.
